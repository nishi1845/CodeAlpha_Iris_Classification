# CodeAlpha_Iris_Classification
A Machine Learning model built using Python and Scikit-learn to classify Iris flower species (Setosa, Versicolor, and Virginica) based on sepal and petal measurements. Includes data visualization using Seaborn and model performance evaluation. Part of the CodeAlpha Data Science Internship program.
# Iris Flower Classification 🌸

This repository contains my submission for **Task 1: Iris Flower Classification** as part of the Data Science Internship at **CodeAlpha**.

## 📌 Project Overview
The goal of this project is to train a machine learning model that can accurately predict the species of an Iris flower based on specific physical measurements: sepal length, sepal width, petal length, and petal width.

## 🛠️ Technologies Used
* **Language:** Python
* **Libraries:** Pandas, Scikit-learn, Matplotlib, Seaborn

## 📊 Workflow
1. **Data Loading:** Using the standard Iris dataset provided by Scikit-learn.
2. **Exploratory Data Analysis (EDA):** Visualizing clusters and relationships between features using Seaborn scatter plots.
3. **Model Training:** Splitting data into 80% training and 20% testing sets, then training a Logistic Regression classifier.
4. **Evaluation:** Assessing performance using accuracy score and classification reports.

## 🚀 How to Run
1. Clone this repository.
2. Install dependencies: `pip install pandas scikit-learn matplotlib seaborn`
3. Run the Jupyter Notebook or Python file inside VS Code.
